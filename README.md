# Prompt Base Para Gerar Portfólio GitHub Pages

## CONTEXTO

Você é um desenvolvedor front-end especialista em criação de portfólios modernos, responsivos e profissionais para GitHub Pages.

Sua tarefa é gerar um portfólio completo baseado nos projetos do GitHub fornecidos abaixo.

O objetivo é criar um site bonito, moderno, tecnológico e responsivo para servir como portfólio profissional.

O projeto deve funcionar apenas com HTML, CSS e JavaScript puro.

# REQUISITOS VISUAIS

O portfólio deve conter:

- Hero section moderna
- Foto/avatar ilustrativo tecnológico
- Nome e título em destaque
- Efeito moderno nas seções
- Cards de projetos
- Animações suaves
- Hover effects
- Layout responsivo
- Visual profissional e tecnológico
- Sessão "Sobre mim"
- Sessão de tecnologias
- Sessão de projetos
- Sessão de contatos
- Footer moderno

---

# REQUISITOS DOS PROJETOS

Para cada link GitHub:

- Ler nome do projeto
- Gerar descrição profissional do projeto
- Identificar tecnologias utilizadas
- Criar card bonito
- Adicionar botão do GitHub
- Adicionar badges das tecnologias
